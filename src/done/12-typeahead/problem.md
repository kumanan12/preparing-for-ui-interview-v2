# Typeahead

**Difficulty**: `Medium`

## Goal
Implement a typeahead (autocomplete) component that filters a list of items based on user input and allows selection.

## Requirements

### Core Functionality
1.  **Input**: Text input field.
2.  **Filtering**: Filter suggestions as user types.
3.  **Dropdown**: Show suggestions in a dropdown list.
4.  **Selection**: Update input value on selection.
5.  **Highlighting**: Highlight matches in the suggestions.

### Accessibility (A11y)
1.  **Role**: Use `combobox` pattern.
2.  **Navigation**: Arrow key navigation support.
